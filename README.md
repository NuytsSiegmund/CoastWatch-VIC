# CoastWatch Victoria

Interactive platform for Victorian coastal monitoring data, integrating UAV photogrammetry with wave measurements across the Victorian Coastal Monitoring Program (VCMP) network.

**[View Platform](https://nuytssiegmund.github.io/CoastWatch-VIC/)** | **[VCMP Website](https://www.marineandcoasts.vic.gov.au/coastal-programs/victorian-coastal-monitoring-program)**

---

## About

CoastWatch Victoria translates coastal monitoring data into actionable insights for coastal managers, researchers, and communities. The platform demonstrates how systematic UAV monitoring and wave analysis can inform decision-making across Victoria's coastline.

**Current Coverage:**
- **44 VCMP sites** visualized on interactive map
- **Port Fairy** detailed analysis (7 years, 54 surveys, 220 transects)
- Framework scalable to all VCMP monitoring sites

## Features

### Victoria Overview
- Interactive map of all 44 VCMP sites across Victoria
- Statewide monitoring statistics and program information
- Site status and survey coverage

### Port Fairy Analysis (Proof-of-Concept)
- 7 years of UAV monitoring data (2018-2025)
- Wave-morphology correlation analysis
- Interactive transect-level elevation change mapping
- Time series visualization and cross-shore analysis
- Wave rose and directional distribution

## Technology

**Data Collection:** DJI Phantom 4 RTK UAVs, Sofar wave buoys  
**Visualization:** Leaflet.js, Chart.js, interactive web platform
