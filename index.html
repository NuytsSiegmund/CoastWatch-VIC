The Ocean's Influence

Wave Monitoring: Real-time data from Sofar Spotter buoy (~3 km offshore, 28m depth)
Data Source: AusWaves - Victorian Wave Network
Observations: 111,991 wave measurements since September 2018

Wave Conditions Drive Beach Change

Offshore wave heights directly control whether beaches erode or build up. Tracking wave conditions over the same period reveals the forcing behind coastal change patterns.

The Connection: Statistical analysis shows a strong relationship (r = 0.61, p < 0.001) between wave height and beach change. Higher waves cause erosion, while calmer periods allow beaches to recover through sand accumulation.

Analytical Methodology

Data Collection

Beach topography surveyed using DJI Phantom 4 RTK UAV at regular intervals from March 2018 to May 2025. Structure-from-Motion photogrammetry processed in Agisoft Metashape to generate high-resolution Digital Elevation Models (DEMs).

Analysis Framework

  • Mean Elevation Change (MEC): Average elevation difference between consecutive surveys across all measurement points
  • Beachface Cluster Dynamics (BCD): Jenks natural breaks classification of morphological states (erosional/neutral/accretional)
  • Residual BCD (r-BCD): Cumulative behavioral index characterizing overall regime of each transect
  • Cross-shore Zonation: Division into offshore, mid-beach, and onshore zones based on elevation profiles
  • Wave Correlation: Integration with offshore wave buoy data (Hsig, Tp, Dp) for forcing-response analysis

Platform Overview

CoastWatch Victoria demonstrates an integrated approach to coastal monitoring data analytics, combining high-resolution UAV-derived topographic surveys with offshore wave measurements. The platform analyzes mean elevation change (MEC), behavioral classification using Jenks natural breaks, and cross-shore zonation to understand coastal vulnerability patterns. Wave-morphology correlation analysis reveals wave height as the primary control on beach change (r = 0.61, p < 0.001), with distinct responses across offshore, mid-beach, and onshore zones. This methodology framework is designed for scalable deployment across all Victorian Coastal Monitoring Program sites, enabling statewide assessment of coastal change patterns and climate adaptation planning.

Methodology Highlights

  • UAV Photogrammetry: DJI Phantom 4 RTK surveys processed with Structure-from-Motion techniques
  • Temporal Analysis: Mean Elevation Change (MEC) calculated between 54 consecutive survey periods
  • Behavioral Classification: Jenks natural breaks algorithm applied to morphological state transitions
  • Cross-shore Zonation: Beach profiles divided into offshore, mid-beach, and onshore zones
  • Wave Integration: Correlation analysis with offshore Sofar Spotter buoy data (111,991 observations)
  • Statistical Validation: Pearson correlation coefficients and significance testing for wave-morphology relationships

Future Development

This platform serves as a proof-of-concept for statewide VCMP data analytics. Future expansion will include additional monitoring sites across Victoria, automated data pipelines for real-time updates, predictive modeling integration, and comparative site analysis tools to support evidence-based coastal management decision-making.
